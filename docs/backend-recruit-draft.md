# [모집] 논문용어사전 백엔드 개발자 모집

## 프로젝트 소개
"논문용어사전"은 논문을 읽을 때 자주 막히는 통계·연구방법론 학술용어를 비전공자도 이해하기 쉽게 풀어 설명하는 사이트입니다. 현재는 정적 사이트(HTML/JS, 서버 없음)로 용어 180여 개와 PDF 논문 뷰어(pdf.js 기반, 클라이언트 처리)를 제공하고 있으며, 서비스를 한 단계 키우기 위해 백엔드를 처음부터 구축해줄 개발자를 찾습니다.

## 담당 업무
- **논문 뷰어 서버 기능**: 논문 업로드/저장, 서버 사이드 파싱, 용어 자동 매칭 API 구축 (현재 클라이언트 전용 pdf.js를 서버 연동으로 확장)
- **회원/검색 API + DB 설계**: 회원가입·로그인, 용어 검색, 즐겨찾기/북마크 기능을 위한 API 서버 및 데이터베이스 설계·구축
- **콘텐츠 자동화 파이프라인**: 논문·용어 데이터를 수집·가공해 terms.json 등 콘텐츠 소스에 자동 반영하는 배치/스크립트 작업
- **애드센스·트래픽 인프라**: 광고 서빙, 방문 로그 수집, 트래픽/사용자 행동 분석을 위한 백엔드 구성

## 이런 분을 찾습니다
- Node.js 또는 Python 기반 API 서버 개발 경험
- RDB/NoSQL 설계 및 운영 경험 (PostgreSQL, MongoDB 등)
- 정적 사이트 → 동적 서비스 전환 경험 있으면 우대
- PDF 파싱/텍스트 처리 경험 있으면 우대
- 1인/소규모 프로젝트를 스스로 기획하고 끝까지 만들어본 경험 우대
